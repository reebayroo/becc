# BECC

## Software Requirement 
* node 0.12+
* go 1.6+
* npm 2.14+
* bower 1.7+
* heroku toolkit
* gulp 3.9+

## How to run locally
```
go get github.com/reebayroo/becc/cmd/exchange/...
go install github.com/reebayroo/becc/cmd/exchange
heroku local
```

## Run tests
```
gulp test
```

## Watch tests
```
gulp watch-test
```

## Run JSLint
```
gulp 
```
